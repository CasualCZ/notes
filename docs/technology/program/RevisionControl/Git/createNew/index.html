    创建版本库 #

    1. 创建一个空目录

       $ cd /usr/local/src
 $ mkdir project

    2. 创建版本库

      • 初始化

          $ git init
  Initialized empty Git repository in /usr/local/src/project/.git/

      • 克隆现有仓库

          $ git clone [url]

    3. 添加文件

       $ git add README.md
 $ git commit -m 'first commit'
