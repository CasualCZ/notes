    ELK Stack #

    ELK是一个实现可靠，安全地从任何来源，任何格式和 实时搜索，分析和可视化的工具

    架构 #

    ELK Stack主要由四个组件组成：

    • Filebeat: 从客户端收集日志并传送给Logstash

    • Logstash: 用于处理传入日志并传送给ElasticSearch

    • ElasticSearch: 存储日志并供Kibana查询

    • Kibana: 用于搜索和可视化日志的Web界面
