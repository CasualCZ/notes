    Kubernetes #

    k8s是一个开源系统，它可以被用于自动部署，扩展和管理容器化（containerized）应用程序。

    架构 #

    Master #

    • etcd

    • API Server

    • Scheduler

    • Controller manager

    Node #

    • Kubelet

    • Kube-proxy

    • cAdvisor

    安装 #

    • Kubeadm

      基于kubeadm工具部署kubernetes,包含Master和Node。(测试版本，不适用于生产环境)

    • Master-Cluster

      基于kubernetes安装包部署kubernetes集群环境。
