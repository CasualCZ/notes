## Git简介

git是一个分布式版本控制软件，最初由林纳斯·托瓦兹（Linus Torvalds）创作，于2005年以GPL发布。最初目的是为更好地管理Linux内核开发而设计。

* 特点

	* 速度

	* 简单的设计

	* 对非线性开发模式的强力支持（允许成千上万个并行开发的分支）

	* 完全分布式

	* 有能力高效管理类似 Linux 内核一样的超大规模项目（速度和数据量）

* 三种状态[工作区域]
	
	* 已提交(commit),[Git仓库]

	* 已修改(modified),[工作目录]

	* 已暂存(staged),[暂存区域]

* 基本工作流程

	1. 在工作目录中修改文件

	2. 暂存文件，将文件的快照放入暂存区域。

	3. 提交更新，找到暂存区域的文件，将快照永久性存储到 Git 仓库目录。
